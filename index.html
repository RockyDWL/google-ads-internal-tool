Project Overview
🛠 Development Status: This tool is currently in active development. The dashboard interface has been built and is shown below as a UI prototype. Google Ads API integration is the next development phase. All data shown in the dashboard is sample/placeholder data used for interface design purposes.
✅ Compliance Summary: This tool is a personal, internal, non-commercial project. It uses read-only API access to retrieve campaign data from a single Google Ads account owned by the developer. No external users. No data sharing. No automated bidding or campaign management.
What This Tool Does

This is a personal dashboard being built by Wenlong Du to view and analyze Google Ads campaign performance data. The tool is designed exclusively for personal use to monitor the developer's own advertising campaigns.

The tool will retrieve campaign metrics from the Google Ads API using read-only access and display them in a private local dashboard. It does not automate any advertising actions.

✅ What This Tool Is
  • Personal internal reporting dashboard
  • Read-only campaign performance viewer
  • Single-developer, single-account tool
  • Private local application (no public access)
  • Manual analysis aid for the developer
❌ What This Tool Is Not
  • Not a SaaS product or public service
  • Not a campaign management or automation tool
  • Not accessible to external users or clients
  • Does not modify bids, budgets, or campaigns
  • Does not access third-party accounts

API Usage Scope
All planned API requests are read-only. This tool will not create, modify, pause, or delete any campaigns, ads, bids, or budgets. It will only query data from a Google Ads account personally owned and operated by Wenlong Du.
Google Ads API Services Planned
  • GoogleAdsService — campaign & ad group metrics reporting
  • CampaignService — read campaign status & settings
  • AdGroupService — ad group performance data
  • All queries use GAQL (Google Ads Query Language)
Metrics to be Retrieved
  • Impressions & clicks
  • Cost and spend data
  • Click-through rate (CTR)
  • Conversions and conversion rate
  • Keyword and ad group performance
  • Campaign status and settings
Example GAQL Query (Read-Only Reporting)
A reporting query this tool will use: 
SELECT
  campaign.id,
  campaign.name,
  campaign.status,
  metrics.impressions,
  metrics.clicks,
  metrics.ctr,
  metrics.cost_micros,
  metrics.conversions,
  metrics.cost_per_conversion
FROM campaign
WHERE segments.date DURING LAST_30_DAYS
ORDER BY metrics.cost_micros DESC

OAuth 2.0 Authorization
Planned Authorization Flow
Access to Google Ads data will be authorized through the official Google OAuth 2.0 flow using the https://www.googleapis.com/auth/adwords scope. Only the Google Ads account owned and managed by Wenlong Du will be authorized to connect with this tool.
Developer (Account Owner) → OAuth 2.0 Auth → Google Ads API → Personal Account Data → Local Dashboard

OAuth2 client credentials and access tokens will be stored server-side only. They will never be exposed in frontend code, browser storage, or shared with any external party.
Security & Data Usage
Planned Security Measures
  • OAuth2 authentication for all API access — credentials stored server-side only
  • Dashboard accessible only to the developer (no public access)
  • No public-facing API endpoints
  • Access tokens never persisted in browser storage
  • Only accounts personally owned by Wenlong Du will be accessed
Data Usage Policy
  • Data used solely for internal reporting and personal analysis
  • No advertising data shared with any third party
  • No personal user data stored or processed
  • No data sold, licensed, or transferred externally
  • Fully compliant with Google Ads API Terms of Service
