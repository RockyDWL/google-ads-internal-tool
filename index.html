About This Tool
What this tool does
This tool is a personal internal reporting tool used only by the developer to analyze performance of Google Ads accounts owned and managed by the developer.

The tool performs read-only requests through the Google Ads API and does not create, modify, or manage campaigns, ads, bids, or budgets.

The tool runs on a private local server. It is not a SaaS product, not a public service, and is not accessible to any external users or clients.

API Usage Scope
All API requests are read-only. This tool does not create, modify, or delete any campaigns, ads, bids, or budgets. It only queries data from Google Ads accounts personally owned by Wenlong Du.
Google Ads API Services Used
  • GoogleAdsService — campaign & ad group metrics reporting
  • CampaignService — read campaign status & settings
  • KeywordPlanIdeaService — keyword performance data
  • All queries use GAQL (Google Ads Query Language)
Metrics Retrieved
  • Impressions & clicks
  • Cost and spend data
  • Click-through rate (CTR)
  • Conversions and conversion rate
  • Keyword and ad group performance
  • Campaign status and settings
Example GAQL Query (Read-Only Reporting)
A typical reporting query used by this tool: 
SELECT
  campaign.name,
  metrics.impressions,
  metrics.clicks,
  metrics.cost_micros,
  metrics.ctr,
  metrics.conversions
FROM campaign
WHERE segments.date DURING LAST_30_DAYS
ORDER BY metrics.impressions DESC

OAuth 2.0 Authorization
Authorization Flow
Access to Google Ads data is authorized through the official Google OAuth 2.0 flow using the https://www.googleapis.com/auth/adwords scope. Only Google Ads accounts owned and managed by the developer are authorized to connect with this tool.
Developer (Account Owner) → OAuth 2.0 Auth → Google Ads API → Personal Account Data → Local Dashboard

OAuth2 client credentials and access tokens are stored server-side only. They are never exposed in frontend code, browser storage, or shared with any external party.
Security & Data Usage
Security Measures

  • Password-protected dashboard — only accessible to the developer
