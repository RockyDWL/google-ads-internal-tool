About This Tool
✅ This is a private internal Google Ads reporting tool. It is actively used by the developer to monitor campaign performance on a single personal account. The tool retrieves data in read-only mode. It is not a SaaS product and is not available to external users.
What this tool does
This tool is a personal internal reporting tool used only by the developer to analyze performance of Google Ads accounts owned and managed by the developer.

The tool performs read-only requests through the Google Ads API and does not create, modify, or manage campaigns, ads, bids, or budgets. All retrieved data is stored locally and never transmitted to any external server or third party.

The tool runs on a private local server. It is not a SaaS product, not a public service, and is not accessible to any external users or clients.

API Usage Scope
All API requests are read-only. This tool does not create, modify, or delete any campaigns, ads, bids, or budgets. It only queries data from a Google Ads account personally owned by Wenlong Du.
Google Ads API Services Used
  • GoogleAdsService — campaign & ad group metrics reporting
  • CampaignService — read campaign status & settings
  • AdGroupService — ad group performance data
  • All queries use GAQL (Google Ads Query Language)
Metrics Retrieved
  • Impressions & clicks
  • Cost and spend data
  • Click-through rate (CTR)
  • Conversions and conversion rate
  • Keyword and ad group performance
  • Campaign status and settings
Example GAQL Query (Read-Only Reporting)
A typical reporting query used by this tool: 
SELECT
  campaign.id,
  campaign.name,
  campaign.status,
  metrics.impressions,
  metrics.clicks,
  metrics.ctr,
  metrics.cost_micros,
  metrics.conversions,
  metrics.cost_per_conversion
FROM campaign
WHERE segments.date DURING LAST_30_DAYS
ORDER BY metrics.cost_micros DESC

OAuth 2.0 Authorization
Authorization Flow
Access to Google Ads data is authorized through the official Google OAuth 2.0 flow using the https://www.googleapis.com/auth/adwords scope. Only the Google Ads account owned and managed by Wenlong Du is authorized to connect with this tool.
Developer (Account Owner) → OAuth 2.0 Auth → Google Ads API → Personal Account Data → Local Dashboard

OAuth2 client credentials and access tokens are stored server-side only. They are never exposed in frontend code, browser storage, or shared with any external party.
Security & Data Usage
Security Measures
  • OAuth2 authentication for all API access — credentials stored server-side only
  • Password-protected dashboard — only accessible to the developer
  • No public-facing endpoints — runs on private local server only
  • Access tokens never persisted in browser storage
  • Only accounts personally owned by Wenlong Du are accessed
Data Usage Policy
  • Data used solely for internal reporting and analysis
  • No advertising data shared with any third party
  • No personal user data stored or processed
  • No data sold, licensed, or transferred externally
  • Fully compliant with Google Ads API Terms of Service
