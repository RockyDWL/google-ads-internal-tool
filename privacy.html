Privacy Policy

Effective Date: March 1, 2026  ·  Last Updated: March 2026

1. Overview

This Privacy Policy describes how Wenlong Du ("I", "me", or "developer") collects, uses, and handles information when operating the RockyDWL Google Ads Internal Management Tool ("the Tool").

This is a personal, non-commercial, internal tool used exclusively by the developer. There are no public-facing users and no third-party access to the Tool.

2. Data Collected

The Tool accesses the following data via the Google Ads API, exclusively from advertising accounts owned and operated by the developer:

  • Campaign performance metrics (impressions, clicks, CTR, cost)
  • Keyword and ad group performance data
  • Advertising budget and spend information
  • Conversion data

No personal data from third parties is collected. No user registration, login credentials, or personally identifiable information from external users is ever collected, because there are no external users.

3. How Data Is Used

All retrieved advertising data is used solely for:

  • Personal campaign performance reporting
  • Budget tracking and spend monitoring
  • Manual optimization of the developer's own advertising campaigns
  • Internal analysis of advertising performance

Data is never used for any commercial purpose, advertising targeting, or resale.

The tool accesses Google Ads campaign performance data through the official Google Ads API in accordance with Google Ads API policies.

4. Data Storage & Security

All data retrieved from the Google Ads API is stored on a private local server accessible only to the developer. OAuth2 credentials and access tokens are stored server-side and are never exposed in frontend code or shared externally.

Access to the dashboard is password-protected and restricted to the developer only.

5. Data Sharing

Data retrieved through this Tool is never sold, shared, transferred, licensed, or disclosed to any third party under any circumstance. There are no advertising partners, analytics services, or third-party integrations that receive any data.

6. Cookies & Tracking

This website does not use cookies, tracking pixels, or any analytics tools. No visitor data is collected from this informational website.

7. Google API Services

This Tool uses the Google Ads API and complies with the Google Ads API Terms of Service and Google API Services User Data Policy.

8. Changes to This Policy

The developer reserves the right to update this Privacy Policy at any time. Changes will be reflected on this page with an updated effective date.
