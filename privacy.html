Privacy Policy

Effective Date: March 1, 2026  ·  Last Updated: March 2026
✅ This tool is a private reporting application operated by a single developer (Wenlong Du) for managing and monitoring the developer's own Google Ads campaigns. It is not a public product. Only data from Google Ads accounts owned and operated by the developer is accessed. No data from external users or third-party clients is processed.

1. Overview

This Privacy Policy describes how Wenlong Du ("the developer") collects, uses, and handles information when operating the RockyDWL Google Ads Reporting Tool ("the Tool").

This is a private, non-commercial reporting tool used by the developer to monitor campaign performance in a Google Ads account owned and operated by the developer. The tool connects to the Google Ads API using OAuth 2.0 and retrieves data in read-only mode.

2. Data Collected

The Tool retrieves the following data via the Google Ads API, exclusively from advertising accounts owned and operated by the developer:

  • Campaign names, IDs, and status
  • Campaign performance metrics (impressions, clicks, CTR, cost)
  • Ad group names and performance data
  • Keyword performance data
  • Advertising budget and spend information
  • Conversion data

Only data from Google Ads accounts owned and operated by the developer is accessed. The tool does not collect any personal information from external individuals, and no third-party client accounts are connected or processed.

3. How Data Is Used

All retrieved advertising data is used solely for:

  • Campaign performance reporting and dashboard display
  • Budget tracking and spend monitoring
  • Analysis of advertising performance trends
  • Supporting manual optimization decisions by the developer

Data is never used for commercial purposes, advertising targeting, automated bidding, or resale of any kind.

4. Data Storage & Security

All data retrieved from the Google Ads API is stored in a private local environment controlled exclusively by the developer. No data is uploaded to external servers, cloud databases, or third-party services. The tool runs in a private environment and is not publicly accessible over the internet.

OAuth2 credentials and access tokens are stored server-side and are never exposed in frontend code or shared externally. Access to the dashboard is restricted to the developer only.

5. Data Sharing

Advertising data retrieved through this Tool is never sold, shared, transferred, licensed, or disclosed to any third party. There are no advertising partners, analytics services, or third-party integrations that receive any data from this tool.

6. Cookies & Tracking

This website does not use cookies, tracking pixels, or analytics tools of any kind. No visitor data is collected from this informational website.

7. Google API Services

This Tool uses the Google Ads API in compliance with the Google Ads API Terms of Service and Google API Services User Data Policy.

Google's Privacy Policy governs all authentication and API interactions: https://policies.google.com/privacy

8. Scope of Access

This tool is operated exclusively by Wenlong Du and accesses only Google Ads accounts owned and operated by the developer. No external accounts, third-party clients, or additional users are involved.

Access to this tool is strictly limited to the developer. The tool provides no multi-user functionality, no public login, and no mechanism for external parties to authorize or connect their accounts.

The tool uses OAuth 2.0 to securely authenticate access to the developer's own Google Ads account. No other individuals can authorize or access the tool.

9. Changes to This Policy

The developer reserves the right to update this Privacy Policy at any time. Changes will be reflected on this page with an updated effective date.
