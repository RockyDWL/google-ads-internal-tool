1. Nature of This Tool

The RockyDWL Google Ads Reporting Tool is a private software tool developed and operated by Wenlong Du for the purpose of monitoring and analyzing Google Ads campaign performance via the Google Ads API.

This tool is not a commercial product, not a SaaS platform, and is not offered as a service to any external parties. It exists to support the developer's own campaign analysis and reporting needs.

2. Purpose of the Tool

This tool connects to the Google Ads API via OAuth 2.0 to retrieve campaign performance metrics — including impressions, clicks, cost, CTR, and conversions — from a Google Ads account owned and operated by the developer.

All API access is read-only. The tool does not create, modify, pause, or delete any campaigns, ads, bids, or budgets. It is used exclusively for performance reporting and analysis.

3. Scope of Access

API access is restricted to a single Google Ads account owned and managed by Wenlong Du. The tool is not used to access accounts belonging to any third party. Access is authorized through a standard Google OAuth 2.0 flow with the https://www.googleapis.com/auth/adwords scope.

4. Prohibited Uses

The following uses are strictly prohibited:

  • Accessing any Google Ads account other than the developer's own
  • Creating, modifying, pausing, or deleting campaigns, ad groups, keywords, or bids
  • Providing access to the tool or its data to any third party
  • Reselling, sublicensing, or commercializing the tool or its outputs

5. Google Ads API Compliance

All use of the Google Ads API through this Tool complies with the Google Ads API Terms of Service and Google's Developer Policies. Specifically:

  • Read-only API access — no campaigns, bids, or budgets are created or modified
  • OAuth2 authentication with standard adwords scope
  • Access restricted exclusively to accounts owned by the developer
  • All data queries performed using GAQL (Google Ads Query Language)

6. Data Handling

All advertising data retrieved via the Google Ads API belongs to the developer's own accounts and is stored locally. No data is shared with third parties. The tool does not process or store personal data of any kind from external individuals.
